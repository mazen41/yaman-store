## PHPMailer Installation

This directory contains manually installed PHPMailer files.

To install PHPMailer properly:

1. Download PHPMailer from GitHub: https://github.com/PHPMailer/PHPMailer/releases
2. Extract the files
3. Copy the following files to this directory:
   - src/PHPMailer.php
   - src/SMTP.php
   - src/Exception.php

Or install using Composer:
```
composer require phpmailer/phpmailer
```
